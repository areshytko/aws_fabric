
## About

A set of fabrics to manage AWS EC2 intances

## Dependencies:

* fabric
* boto3
* aws cli

## Installation:

pip install boto3 fabric aws

[configure AWS credentials](https://boto3.readthedocs.io/en/latest/guide/quickstart.html#configuration)
